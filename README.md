# Claude-Skills · 通用技能库

Claude Code通用Skill库，适用于所有项目。

## 使用方式
电脑到了，告诉Claude Code：
「读Claude-Skills里的所有Skill，按照规范开发项目」

## Skill列表（Claude Code自动生成）
等电脑到了，运行以下命令让Claude Code生成所有Skill：

「帮我建一个通用Claude Code Skill库，覆盖以下方向：

后端开发规范（FastAPI）：
- 项目结构规范
- API接口设计规范
- 错误处理规范
- 数据库操作规范（SQLAlchemy）
- 认证授权规范（JWT）
- 日志记录规范
- 测试规范

前端开发规范（Next.js）：
- 项目结构规范
- 组件设计规范
- 状态管理规范
- API请求规范
- 错误处理规范
- 响应式设计规范
- 性能优化规范

数据库规范（PostgreSQL）：
- 表结构设计规范
- 索引优化规范
- 迁移规范
- 备份规范

部署规范：
- 环境变量管理
- Docker配置
- CI/CD规范
- 监控告警

安全规范：
- 输入验证
- SQL注入防护
- XSS防护
- CSRF防护
- 密码加密（bcrypt）
- API限流

代码质量规范：
- 命名规范
- 注释规范
- Git提交规范
- Code Review规范

每个Skill都要：
- 详细可执行的规则
- 示例代码
- 注意事项」
